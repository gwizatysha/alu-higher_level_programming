# Python - More Data Structures: Sets, Dictionaries

This project covers more advanced Python data structures: sets, dictionaries, and using `map` for functional-style list operations.

## Tasks

0. square_matrix_simple - Compute the square of all integers in a matrix
1. search_replace - Replace all occurrences of an element in a new list
2. uniq_add - Add all unique integers in a list
3. common_elements - Return a set of common elements in two sets
4. only_diff_elements - Return a set of elements present in only one set
5. number_keys - Return the number of keys in a dictionary
6. print_sorted_dictionary - Print a dictionary ordered by keys
7. update_dictionary - Replace or add a key/value in a dictionary
8. simple_delete - Delete a key in a dictionary
9. multiply_by_2 - Return a new dictionary with all values multiplied by 2
10. best_score - Return the key with the biggest integer value
11. multiply_list_map - Multiply all values in a list using map
12. roman_to_int - Convert a Roman numeral to an integer
