# Python - Exceptions

This project covers exception handling in Python using try/except/finally, and how to raise custom exceptions.

## Tasks

0. safe_print_list - Print x elements of a list safely
1. safe_print_integer - Safely print an integer using try/except
2. safe_print_list_integers - Print and count only integers in a list
3. safe_print_division - Divide 2 integers safely with a finally block
4. list_division - Divide elements of two lists safely
5. raise_exception - Raise a TypeError exception
6. raise_exception_msg - Raise a NameError exception with a message
