# Python - Data Structures

This project covers basic Python data structures: lists, tuples, and manipulating them without relying on certain built-in functions or modules.

## Tasks

0. print_list_integer - Print all integers of a list
1. element_at - Retrieve an element from a list safely
2. replace_in_list - Replace an element in a list
3. print_reversed_list_integer - Print list integers in reverse
4. new_in_list - Replace an element in a copy of a list
5. no_c - Remove c and C characters from a string
6. print_matrix_integer - Print a matrix of integers
7. add_tuple - Add two tuples
8. multiple_returns - Return length and first character of a string
9. max_integer - Find the max integer in a list
10. divisible_by_2 - Find multiples of 2 in a list
11. delete_at - Delete an item at a specific index
12. switch - Switch values of two variables
